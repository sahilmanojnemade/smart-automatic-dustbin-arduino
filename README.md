This project is a Smart Automatic Dustbin developed using Arduino, designed to provide a touch-free waste disposal system. The dustbin automatically opens its lid when a hand or object is detected near the sensor, improving hygiene and convenience.

The system uses an ultrasonic sensor for distance detection and a servo motor to control the lid movement.
